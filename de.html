DeComputerParts

DeComputerParts is the perfect tool for managing your computer tools and accessories. With our easy-to-use interface, you can add new tools, manage payments, and more. Plus, it's payment integration via Stripe makes it easy to get started.

What Technology I used this project?
  1. Used Tailwind css & Daisy ui for my entire project design
  2. JWT for user verification and also user access to my item
  3. Custom admin function for admin access
  4. Stripe for Payment Integation
  5. Authentication like Log in, Sign Up, I used firebase
